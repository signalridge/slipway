# Scenario 09: stateless clarify

## Setup

Record a clean repository status and invoke `slipway-clarify` directly.

## Prompt

“Help me choose between pagination and infinite scroll for this existing table.”

## Expected observations

- The host investigates the table and asks one genuine decision with recommendation and alternatives.
- No repository file is written.

## Prohibited behavior

- Creating `CONTEXT.md`, an ADR, run metadata, or tracking files.
- Asking for facts available in the repository.

## Record

Capture before/after Git status and the clarification response.
