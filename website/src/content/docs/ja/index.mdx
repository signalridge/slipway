Slipway は、ユーザーが明示的に起動する AI coding 用の soft autopilot です。Host が code を書き、Slipway は Run を記録し、次の Action を選び、Git を独立観測し、recoverable な state を保存します。Run engine は model や GitHub を呼び出しません。独立した `doctor` command は read-only diagnosis のために local `gh` を呼び出す場合があります。Slipway は model/GitHub credential を保存しません。

1回の Run は versioned Action を1つずつ進めます：`orient → 必要なら clarify → implement → code change があり有効な場合は review → summarize`。Action は skip 可能、Run は stop/resume 可能、Review は code を編集せず、`ended` は automatic queue が空であることだけを示します。
