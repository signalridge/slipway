日本語は non-normative summary です。完全な中国語製品契約と versioned machine schema が実装 authority です。Slipway は explicit invocation のみで開始し、issue-first ですが GitHub を gate にせず、ad-hoc escape hatch を残します。Self-contained Change だけが issue-backed source で、Objective は任意 planning input、実行不可です。

Run は1つずつ versioned Action を返します：`orient → 必要なら clarify → implement → observed diff なら review → summarize`。Action は skip、Run は stop/resume、Review は repair せず、`ended` は automatic queue が空であることだけを示します。
