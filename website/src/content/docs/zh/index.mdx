Slipway 是一款由用户显式调用的 AI 编程软自动驾驶。宿主写代码；Slipway 记录每次 Run、选择下一个 Action、独立观察 Git，并保存可恢复状态。Run engine 不调用模型或访问 GitHub；独立的 `doctor` 命令可能调用本机 `gh` 做只读诊断。Slipway 不保存模型或 GitHub 凭据。

一次 Run 每次推进一个版本化 Action：`orient → 必要时 clarify → implement → 代码已变化且启用时 review → summarize`。每项 Action 可以 skip，Run 可以 stop/resume，Review 不改代码，`ended` 只表示自动队列为空。
