完整中文产品契约与版本化 machine schema 是实现权威。Slipway 只在显式要求后启动，issue-first 但不把 GitHub 变成 gate，并保留 ad-hoc escape hatch。自包含 Change 是唯一 issue-backed source；Objective 只是可选 planning input，不能执行。

Run 每次返回一个版本化 Action：`orient → 必要时 clarify → implement → 观察到 diff 时 review → summarize`。每项可 skip，Run 可 stop/resume，Review 不修复，`ended` 只表示自动 queue 为空。
