English is a non-normative summary. The complete Chinese product contract and versioned machine schema are implementation authorities. Slipway starts only when explicitly asked, is issue-first without making GitHub a gate, and preserves an ad-hoc escape hatch. A self-contained Change is the only issue-backed source; an Objective is optional planning input and never executable.

One Run returns one versioned Action at a time: `orient → clarify when needed → implement → review on observed diff → summarize`. Every Action can be skipped, a Run can stop/resume, Review never repairs, and `ended` means only that the automatic queue is empty.
